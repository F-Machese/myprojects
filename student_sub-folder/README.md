# student_sub-folder
This folder contains all files related to the analysis of the Student_performance_10k.csv dataset obtained from `Kaggle`.

## Contents
libraries.txt - List of Python libraries required to run the analysis scripts. Install using pip install -r libraries.txt.
Student_performance_10k.csv - Original raw dataset containing student performance and demographic information.
student_performance_cleaned.csv - Cleaned version of the dataset.
student_performance.py - Python script containing all analysis: data cleaning and transformation, descriptive statistics, inferential statistics tests (t-tests, ANOVA and chi-square), anomaly detection and visualizations.
